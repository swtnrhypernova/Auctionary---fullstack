# Auctionary---fullstack
Auctionary is a web project where users can sell and bid on auctions.

Languages used:
-	Frontend: HTML5, CSS3, Vue.js, Tailwind CSS
- Backend: Node.js, Express.js, RESTful APIs
-	Databases: SQLite, SQL
-	Tools: Git, GitHub, VS Code, Postman

More information:
-	Designed and developed a complete auction-based web application allowing users to create listings and place bids on items
-	Built the entire frontend and backend using JavaScript, Node.js, Express, Vue.js, and SQLite
-	Implemented secure user authentication with robust registration and login validation
-	Integrated a profanity filter to moderate user-generated content
-	Developed item categorisation to improve search and browsing functionality
-	Applied MVC principles to maintain a clean, scalable, and maintainable codebase
